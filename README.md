# Oldiestore, a basic ecommerce website for selling video games

## Requirement:
* NodeJS 14+
* Yarn (Recommmended) / NPM

## Running: 
### Run the project by this command:
* Client

```
    cd ./client
    yarn instart
    yarn start
```
* API

```
    cd ./client
    yarn instart
    yarn start
```
* Admin

```
    cd ./client
    yarn instart
    yarn start
```
